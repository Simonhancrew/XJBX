这里叙述一下基本的实现的思路

![2](./pic/2.png)

1. 注册中心是首要，可以考虑使用zk。注册中心放着服务的地址，主要负责注册和查找，可以理解成一个目录，服务端启动的时候会将服务名+ （ip,port）注册到注册服，消费者根据服务名称找到具体的服务地址。然后可以通过网络请求去请求服务端
